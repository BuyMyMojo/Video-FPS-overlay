# Video FPS overlay
 Create FPS graph overlays like TRDrop and Digital Foundry using FrameView, Mangohud and MSI Afterburner


### Features

- PNG overlay export
- Frame time to FPS conversion
- Nvidia FrameView support

### WIP

- Mangohud Support
- MSI Afterburner support
- Jpeg export
- Auto overlay on video
